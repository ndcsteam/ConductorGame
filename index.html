傾斜 Micro:bit 來移動點的位置

使用說明:

  1. 點擊「連接 Micro:bit」按鈕
  2. 從彈出的藍牙設備列表中選擇您的 Micro:bit
  3. 連接後，傾斜 Micro:bit 來查看位置變化
  4. X 值代表左右傾斜，Y 值代表前後傾斜

故障排除:

  • 請確保 Micro:bit 已啟動並運行您的程式
  • 確認藍牙已在您的裝置上啟用
  • 如果連接失敗，請嘗試重啟 Micro:bit
  • 本應用需要使用 Chrome、Edge 或 Opera 等支援 Web Bluetooth API 的瀏覽器
  • 連接問題可在瀏覽器控制台（F12）中查看詳細錯誤信息
